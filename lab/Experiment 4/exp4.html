Part 1: Flask App

mkdir my-flask-app cd my-flask-app from flask import Flask app = Flask(__name__) @app.route('/') def hello(): return "Hello from Docker!"

Dockerfile

FROM python:3.9-slim WORKDIR /app COPY requirements.txt . RUN pip install -r requirements.txt COPY app.py . EXPOSE 5000 CMD ["python", "app.py"]

.dockerignore

__pycache__/ .env .vscode/ .git

Build Commands

docker build -t my-flask-app . docker images docker tag my-flask-app:latest myapp:v1

Run Container

docker run -d -p 5000:5000 my-flask-app docker ps docker logs container-name

Multi-stage Build

FROM python:3.9-slim AS builder RUN pip install -r requirements.txt FROM python:3.9-slim COPY --from=builder /app /app

Docker Hub

docker login docker push username/myapp docker pull username/myapp

Node.js Example

FROM node:18-alpine WORKDIR /app COPY . . RUN npm install CMD ["node", "app.js"]
